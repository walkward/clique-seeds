## Table of Contents


## Install

[npm][]:

```sh
npm install clique-seeds
```

[yarn][]:

```sh
yarn add clique-seeds
```


## Usage

```js
const CliqueSeeds = require('clique-seeds');

const cliqueSeeds = new CliqueSeeds();

console.log(cliqueSeeds.renderName());
// script
```
